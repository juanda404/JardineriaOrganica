# 🌿 Jardinería Orgánica - Landing Page

¡Bienvenido al repositorio de la plataforma web de **Jardinería Orgánica**! Esta es una landing page moderna, rápida y optimizada para dispositivos móviles, diseñada para presentar servicios de mantenimiento, paisajismo y bioinsumos en Jamundí, Cali y todo el Valle del Cauca.

---

## 🚀 Características del Proyecto

* **Diseño Moderno:** Desarrollado utilizando **Tailwind CSS** para una UI fluida, limpia y responsiva.
* **Tarjetas de Servicios Optimizadas:** Integración de imágenes reales utilizando técnicas de recorte dinámico (`object-cover`) para evitar distorsiones.
* **Estructura Semántica:** HTML5 limpio (`<header>`, `<main>`, `<section>`, `<footer>`) para mejorar el rendimiento y SEO básico.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3**
* **Tailwind CSS (v3+)** - CDN de desarrollo.
* **FontAwesome** - Iconografía vectorial para rendimiento óptimo.
